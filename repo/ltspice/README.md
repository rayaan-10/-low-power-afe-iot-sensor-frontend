# LTspice files

Place your `.asc` (schematic source) and `.raw` (simulation output, optional — large file,
can be gitignored) here.

If you can't locate the original `.asc` file, the schematic is fully visible in
`images/schematics/ltspice-full-schematic.png` and can be quickly rebuilt in LTspice —
component values are all readable on that image, and the netlist logic is documented in
`docs/design-notes.md`.
